## 二、世界观设定

### 2.1 时代背景

描述故事发生的时代、地点、社会环境。包括：

- 现实世界基础（现代都市？架空世界？平行时空？）
- 与现实的差异点（修炼者存在？超自然现象？隐世势力？）

### 2.2 特殊规则体系

根据小说类型，定义核心体系：

**修仙类示例：**

```
修炼境界：炼气 → 筑基 → 金丹 → 元婴 → 化神 → 大乘 → 渡劫
每个境界分初期、中期、后期、巅峰四个小阶段
突破方式：悟道 + 资源 + 天劫
特殊能力：随境界提升解锁（神识、飞行、领域等）
```

**兵王类示例：**

```
实力等级：普通士兵 → 特种兵 → 王牌 → 战神 → 兵王
组织体系：暗部、龙组、神盾局等
技能体系：格斗、枪械、潜入、信息战、心理战
```

### 2.3 势力格局

列出主要势力及关系：

```
势力名称 | 实力定位 | 核心人物 | 与主角关系 | 备注
---------|----------|----------|------------|---------
XX家族   | 一线     | 家主XXX  | 初期对手   | 后期盟友
YY门派   | 顶级     | 掌门YYY  | 主角师门   |
ZZ组织   | 暗中势力 | 首领ZZZ  | 终极BOSS   | 伏笔线
```

## 三、主线剧情大纲

### 3.1 核心矛盾

一句话描述贯穿全书的核心冲突：
> 例：废材少年觉醒远古血脉，在步步紧逼的家族阴谋中逆天崛起，最终揭开身世之谜。

### 3.2 主线走向

```
开局（第1卷）：
  - 主角出场方式与初始状态
  - 核心矛盾初现
  - 第一个小高潮/爽点

发展（第2-3卷）：
  - 实力提升路线
  - 势力扩展节奏
  - 敌对升级
  - 关键伏笔铺设

高潮（第4-5卷）：
  - 核心冲突全面爆发
  - 大反转
  - 终极对决

结局（最终卷）：
  - 主线收束
  - 伏笔回收
  - 角色归宿
```

## 四、分卷规划

每卷按以下格式填写：

### 第X卷：卷名

```
核心事件：本卷最重要的一件事
主要冲突：本卷的核心矛盾是什么
预计章节：第XX章 - 第XX章（约XX章）

新登场角色：
  - 角色A（身份，与主角的关系）
  - 角色B（身份，功能定位）

爽点/高潮设计：
  1. 第XX章左右 — 描述爽点内容
  2. 第XX章左右 — 描述高潮场景

本卷主角成长：
  - 实力：从XX到XX
  - 心境：从XX到XX
  - 势力/人脉：获得XX

伏笔安排：
  - 埋设：描述要埋的伏笔
  - 回收：描述要回收的之前伏笔

本卷结尾钩子：如何吊起读者对下一卷的期待
```

## 六、节奏规划

全书情绪曲线参考（1-10分标注紧张度）：

```
卷1：3 → 5 → 7 → 4 → 8（结尾高潮）
卷2：4 → 6 → 5 → 8 → 9（全书第一个大高潮）
卷3：5 → 7 → 6 → 8 → 7
卷4：6 → 8 → 7 → 9 → 10（终极决战）
卷5：7 → 5 → 8 → 6 → 9（收束）
```

每卷内部遵循"小爽 → 铺垫 → 中爽 → 铺垫 → 大爽"的节奏循环。

## 填写指南

1. **先粗后细**：第一遍只填核心骨架，后续逐步补充细节
2. **保持灵活**：大纲是指南不是枷锁，写作过程中允许微调
3. **记录变更**：大纲修改时标注原因和影响范围
4. **定期回顾**：每完成一卷回顾一次大纲，确保整体方向没有偏离
