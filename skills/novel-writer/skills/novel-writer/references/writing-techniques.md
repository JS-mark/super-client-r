# 爽文写作技巧

## 一、爽点设计

### 爽点类型清单

| 类型 | 描述 | 使用频率 |
|------|------|----------|
| 打脸爽 | 被小看 → 当场翻盘，啪啪打脸 | 高频，每1-2章可用 |
| 装逼爽 | 不经意间展露实力，震惊全场 | 中频，每3-5章 |
| 逆袭爽 | 绝境翻盘，越被压越强 | 中频，每卷1-2次 |
| 暴富爽 | 突然获得大量资源/宝物 | 低频，需要铺垫 |
| 护短爽 | 谁动我的人我灭谁 | 中频，读者最爱 |
| 身份揭秘爽 | 隐藏身份曝光，全场震惊 | 低频，杀手锏 |
| 成长突破爽 | 关键时刻突破境界/实力 | 每卷1-2次 |
| 收服爽 | 曾经的对手/强者心服口服 | 中频 |

### 爽点设计公式

```
铺垫（压抑感）→ 触发（爆发点）→ 释放（爽感）→ 余韵（反应/评价）
```

**关键：铺垫越充分，释放越过瘾。不要急着爽，先让读者憋着。**

### 爽点节奏分布

```
一章之内：小爽（一个小冲突小化解）
三章一组：中爽（一个完整的爽文小循环）
一卷之内：大爽（核心矛盾阶段性解决）
全书节点：超爽（改变格局的关键反转）
```

## 二、伏笔设计

### 伏笔的三个层次

- **明线伏笔** — 读者能感觉到"这里有东西"，但猜不到具体是什么
- **暗线伏笔** — 读者完全注意不到，回收时恍然大悟
- **双线伏笔** — 读者以为猜到了真相，实际还有反转

### 伏笔埋设技巧

1. **自然融入** — 伏笔必须像正常叙事的一部分，不能刻意
2. **多次提及** — 重要伏笔至少提2-3次，让读者有印象但不突兀
3. **错位时间** — 埋和收之间至少隔10章以上
4. **关联情感** — 好的伏笔回收时要带情感冲击

### 伏笔管理规范

每埋一条伏笔，立即记录到 `novel/progress.md`：

```
伏笔编号：F-XXX
内容：具体是什么
埋设位置：第X卷第X章
计划回收：第X卷第X章左右
状态：未埋 / 已埋 / 已部分回收 / 已完全回收
```

## 三、节奏控制

### 黄金节奏法则

- **三章一转**：每三章内有一个明显情节转折或爽点
- **五章一潮**：每五章有一个小高潮
- **十章一战**：每十章左右安排一场核心战斗/冲突

### 张弛有度

```
紧 → 紧 → 缓 → 紧 → 缓 → 紧紧紧（大高潮）→ 缓缓 → 紧 ...
```

**缓冲章不水的方法：** 日常中穿插人物塑造、轻松互动中埋伏笔、修炼提升中带悬念、新角色以轻松方式登场。

### 章末钩子类型

| 类型 | 示例 |
|------|------|
| 悬念型 | "门缓缓推开，走进来的人让所有人都变了脸色——" |
| 反转型 | "他以为一切结束了。但真正的麻烦，才刚刚开始。" |
| 情感型 | "她转身走进雨里，始终没有回头。" |
| 爆点型 | "天际炸开一道金光——那是渡劫的天象！" |
| 信息型 | "信封里只有四个字：你父未死。" |

## 四、战斗描写

### 战斗不是记流水账

```
❌ 他出了一拳，对方挡住了。他又踢了一脚，对方闪开了。

✅ 拳锋撕裂空气，带着令人窒息的压迫感。
   对方双臂交叉格挡——晚了半拍。
   咔嚓。清脆的断裂声比惨叫来得更早。
```

### 战斗描写五要素

1. **五感代入** — 声音、气味、触感、温度
2. **节奏变化** — 短句加速紧张感，长句拉慢镜头
3. **心理活动** — 穿插角色判断和情绪
4. **环境互动** — 利用场景元素
5. **旁观者反应** — 侧面烘托战斗级别

## 五、对话写作

### 对话的功能

每句对话至少承担一个功能：推进剧情、塑造人物、传递信息、制造冲突、提供幽默。如果一句对话什么都不做，删掉它。

### 潜台词比台词重要

同一句"随便你"，可以是"我已经不在乎了"、"我在赌气"、"我尊重你但心里难受"。不同角色、不同场景，含义完全不同。

### 避免信息倾倒

```
❌ "我们张家在三百年前由初代家主创立，经过十二代传承……"
✅ "三百年底蕴，你以为是白叫的？" 他嗤笑，"光化神期老祖，我们就有三位。"
```

## 六、场景描写

### 少即是多

```
❌ 房间里有红木桌子，桌上有白色茶壶，旁边三个杯子，墙上挂画……
✅ 茶烟袅袅升起，淡得像随时会散。窗外月色照得桌上信纸发白。
```

### 场景服务情节

- **紧张时**：短句、暗色调、封闭空间
- **轻松时**：长句、明亮色调、开阔空间
- **悲伤时**：灰色调、寂静、冷感
- **爽快时**：强烈对比、金光/火光、围观者震撼
