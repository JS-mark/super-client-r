# 角色档案模板与塑造技巧

## 一、角色档案模板

每个重要角色填写一份完整档案：

### 基本信息

```
姓名：
别名/外号：
年龄：
性别：
外貌特征：（抓2-3个标志性特征，避免流水账式描写）
  - 第一印象：
  - 标志性特征：
  - 穿着风格：
```

### 性格画像

```
核心性格：（用3个关键词概括）
表面性格：（别人眼中的他/她）
真实性格：（内心深处的他/她）
说话风格：（举一个典型台词示例）
口头禅：
行为习惯：（紧张时会做什么？思考时的小动作？）
```

### 背景故事

```
出身：

人生转折点：（改变命运的事件）
进入故事的契机：（为什么出现在这个故事里）
```

### 内心驱动

```
执念/目标：（活着的最大动力是什么）
软肋/弱点：（什么会让他/她崩溃或失控）
恐惧：（最害怕的事情）

```

### 实力与成长

```

每次突破的契机：
  - 第一次：
  - 第二次：
  - 第三次：
标志性技能/绝招：
```

### 人物关系

```
与主角：
与女主/男主：
与反派：
与配角A：
关系变化线：初始关系 → 中期变化 → 最终关系
```

### 反差感设计

```
表面 vs 真实：（如：看似纨绔实则深藏不露）
强处 vs 弱点：（如：战场无敌但不会哄女朋友）
过去 vs 现在：（如：曾是废材如今逆天）
```

### 登场规划

```
首次登场：第X卷第X章

登场时给读者的印象：

```

## 二、角色塑造技巧

### 1. 标签化出场

每个角色首次出场要给读者留下一个鲜明标签：

**反面教材：**
> 一个穿着西装的中年男人走了进来。

**正面示范：**
> 门被推开，一股廉价烟味率先到场。来人西装笔挺，但袖口烧了个洞，像是不小心，又像是故意的——这人身上的每一处破绽，都透着精心设计的味道。

**技巧：** 用一个细节（行为/物件/气质）定义角色，比罗列外貌有效十倍。

### 2. 对话辨识度

不同角色的说话方式应该有明显区别：

```
主角（痞帅型）：   "这事儿，我能说不吗？" 他笑着把烟掐了，"我还真不能。"
军师（儒雅型）：   "此事急不得，容我思量。" 他轻摇折扇，"不过……也急不了多久。"
反派（阴沉型）：   "你以为你赢了？" 他的声音平静得像一潭死水，"我只是还没开始。"
暴脾气配角：      "废什么话！干就完了！"
```

**要点：** 句式长短有差异，语气词有特色，潜台词比表面意思重要。

### 3. 行为定义性格

不要告诉读者"他很善良"，而是通过行为展现：

**告诉（差）：** 他是个善良的人。
**展现（好）：** 他把最后一个馒头推给了乞丐，自己灌了两口凉水充饥，然后笑着说"我正减肥呢"。

### 4. 反差制造魅力

角色身上的矛盾越合理，越有魅力：

- 杀伐果断的男主，遇到小动物就走不动道
- 冷面杀手有个暗搓搓养花的爱好
- 嚣张跋扈的大小姐，在喜欢的人面前结巴
- 看似废柴的男主，关键时刻一句话镇住全场

**关键：** 反差必须有合理的内在逻辑，不能为反差而反差。

### 5. 成长弧线设计

优秀的角色成长不是单纯变强，而是多维度递进：

```
维度1 · 实力：  废柴 → 小有所成 → 一方强者 → 顶尖高手
维度2 · 心境：  莽撞冲动 → 学会隐忍 → 运筹帷幄 → 举重若轻
维度3 · 格局：  只顾自己 → 保护身边人 → 肩负一方 → 心怀天下
维度4 · 人脉：  孤身一人 → 三五兄弟 → 势力雏形 → 一呼百应
```

**每次成长都需要：** 铺垫（为什么能成长）→ 代价（成长不是白来的）→ 回报（成长带来的爽感）

### 6. 配角不是工具人

即使是配角，也要给予基本的人物弧光：有自己的动机、性格、结局。

### 7. 反派塑造

好的反派和主角一样重要：

```
❌ 差的反派：因为他是坏人所以做坏事
✅ 好的反派：他有自己的信念和执念，只是走了和主角相反的路
```

反派档案要额外包含：动机（要合理到读者能共情）、信念（与主角形成对照）、实力来源（有说服力）、结局（有分量）。
