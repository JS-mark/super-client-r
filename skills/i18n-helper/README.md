# i18n-helper -- i18n 多语言助手

将代码中的硬编码 UI 文本替换为 i18n 翻译调用，同步更新 en/zh 语言文件。

## 使用场景

- 开发新功能时，将中文/英文硬编码文本国际化
- 批量处理文件中的未翻译文本
- 确保 en/zh 语言文件保持同步

## 触发方式

### 斜杠命令

```
/i18nify
/i18nify src/renderer/src/components/settings/GeneralSettings.tsx
```

### 自然语言触发

- "把这个文件做多语言化"
- "国际化这段代码"
- "帮我处理 i18n"
- "替换硬编码文本为翻译调用"

## 工作流程

1. 扫描目标文件中的硬编码 UI 文本
2. 根据文件路径确定 i18n namespace
3. 生成语义化的 translation key
4. 展示变更计划，等待用户确认
5. 更新 en/zh 语言 JSON 文件
6. 替换源代码中的硬编码文本为 `t()` 调用
7. 验证改动正确性
