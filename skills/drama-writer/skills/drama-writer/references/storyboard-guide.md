# 分镜场景指南

本指南用于为短剧剧本生成分镜场景描述，帮助导演/摄影师理解编剧的视觉意图。

---

## 1. 常用镜头类型

### 景别分类

| 景别 | 缩写 | 画面范围 | 常用场景 |
|------|------|---------|---------|
| 大远景 | ELS | 广阔环境，人物极小 | 建立场景、展示世界观 |
| 远景 | LS | 全身 + 大量环境 | 场景建立、人物与环境关系 |
| 全景 | FS | 人物全身 | 角色登场、群戏走位 |
| 中景 | MS | 膝盖以上 | 日常对话、动作交代 |
| 中近景 | MCU | 胸部以上 | 双人对话、情绪表达 |
| 近景 | CU | 肩部以上 | 表情细节、情绪特写 |
| 特写 | ECU | 局部特写 | 眼神、手部、关键道具 |

### 短剧景别使用频率

| 平台 | 主力景别 | 偶尔使用 | 谨慎使用 |
|------|---------|---------|---------|
| 竖屏 | 近景、中近景、特写 | 中景 | 全景、远景（画面太小） |
| 横屏 | 中景、中近景、近景 | 全景、特写 | 远景（控制成本） |

> 竖屏短剧因为屏幕纵向比例，远景和全景中人物会非常小，尽量避免。

## 2. 镜头运动

### 基本运动类型

| 运动 | 说明 | 情绪效果 | 示例 |
|------|------|---------|------|
| 推 | 镜头向前移动/变焦推进 | 聚焦、紧张、强调 | 推向角色惊愕的脸 |
| 拉 | 镜头向后移动/变焦拉远 | 揭示、孤独、疏离 | 从特写拉到全景，揭示空旷大厅 |
| 摇 | 镜头左右/上下旋转 | 展示环境、跟随视线 | 摇过一排跪着的人 |
| 移 | 镜头横向平移 | 平稳跟随、并行感 | 跟着角色走过走廊 |
| 跟 | 镜头跟随主体移动 | 代入感、紧张追逐 | 手持跟拍角色奔跑 |
| 升降 | 镜头垂直上升或下降 | 俯视全局/仰视权威 | 从低角度升起，揭示城市全景 |
| 环绕 | 镜头绕主体旋转 | 强调主体、仪式感 | 环绕两人对峙的画面 |

### 特殊运动

| 运动 | 说明 | 常用场景 |
|------|------|---------|
| 甩镜 | 快速摇到另一个主体 | 突然发现、反应镜头 |
| 变焦推/拉 | 焦距变化制造压迫或抽离 | 震惊、顿悟 |
| 手持晃动 | 刻意的不稳定 | 紧张、混乱、追逐 |
| 升格（慢动作） | 高帧率拍摄 | 关键动作、名场面 |
| 降格（快动作） | 低帧率拍摄 | 时间流逝、搞笑 |

## 3. 竖屏 vs 横屏构图差异

### 竖屏短剧（9:16）

**核心原则**: 人物为主，环境为辅。

| 要素 | 竖屏要求 |
|------|---------|
| 人物位置 | 居中偏上，占画面60%以上 |
| 对话构图 | 上下构图（A在上，B在下）或前后景 |
| 环境交代 | 用局部细节暗示环境，避免空旷全景 |
| 文字/字幕 | 底部1/3预留给字幕区 |
| 动作方向 | 上下运动比左右运动更有冲击力 |

**竖屏特有构图**:

```
┌─────────┐
│  留白    │
│ ┌─────┐ │
│ │角色A │ │
│ │表情  │ │
│ └─────┘ │
│         │
│ ┌─────┐ │
│ │角色B │ │
│ │表情  │ │
│ └─────┘ │
│ [字幕区]│
└─────────┘
对话场景：上下分布
```

### 横屏微短剧（16:9）

**核心原则**: 标准电影构图，兼顾人物与环境。

| 要素 | 横屏要求 |
|------|---------|
| 人物位置 | 三分法构图 |
| 对话构图 | 正反打、过肩镜头 |
| 环境交代 | 可用建立镜头交代场景 |
| 群戏 | 可容纳多人互动 |
| 运镜 | 横向运动空间充足 |

## 4. 转场类型

### 常用转场

| 转场类型 | 效果 | 适用场景 | 分镜标注 |
|----------|------|---------|---------|
| 硬切 | 直接跳转 | 快节奏、日常衔接 | `[硬切]` |
| 叠化 | 画面渐变重叠 | 时间流逝、抒情 | `[叠化：2s]` |
| 闪白 | 白光闪烁 | 冲击、觉醒、闪回 | `[闪白]` |
| 闪黑 | 画面骤暗 | 失去意识、时间跳跃 | `[闪黑]` |
| 划变 | 画面滑动替换 | 平行叙事、空间转换 | `[划变：左→右]` |
| 匹配剪辑 | 相似元素衔接 | 创意转场 | `[匹配：眼睛合上→门关上]` |
| 声音先行 | 下一场的声音先出现 | 平滑过渡 | `[声音先行：电话铃]` |
| 动作衔接 | 动作延续到下一场 | 连贯感 | `[动作衔接：转身]` |

### 短剧转场原则

1. **竖屏以硬切为主**: 节奏快，不拖泥带水
2. **闪回用闪白/闪黑**: 明确标记时间跳跃
3. **集末用淡出黑场**: 标准集结束信号
4. **避免花式转场**: 短剧要干净利落，花里胡哨的转场会显得廉价

## 5. 场景气氛营造

### 灯光提示

| 氛围 | 灯光风格 | 适用场景 |
|------|---------|---------|
| 温馨 | 暖色调，柔光 | 甜蜜互动、家庭场景 |
| 紧张 | 侧光，明暗对比强 | 对峙、冲突 |
| 悬疑 | 低调照明，阴影多 | 阴谋、秘密 |
| 压抑 | 冷色调，顶光 | 审讯、控制 |
| 权力 | 逆光剪影 | 大佬登场、权势展示 |
| 浪漫 | 柔焦背光 | 暧昧、告白 |
| 绝望 | 单一光源，大面积暗 | 至暗时刻 |

### 色调提示

| 情绪 | 建议色调 |
|------|---------|
| 日常/中性 | 自然色温 |
| 回忆/闪回 | 偏暖黄或去饱和 |
| 危险/紧张 | 偏冷蓝或偏红 |
| 梦境/幻觉 | 高饱和、偏紫/青 |
| 复仇/阴谋 | 低饱和、冷调 |

## 6. 特效场景标注规范

需要后期特效的场景必须明确标注，便于预算和制作排期。

### 特效分级

| 等级 | 说明 | 示例 | 成本影响 |
|------|------|------|---------|
| S级 | 大型CG场景 | 仙术对决、城市毁灭 | 极高 |
| A级 | 中型特效 | 飞行、变身、大范围法术 | 高 |
| B级 | 小型特效 | 小型法术光效、伤口化妆 | 中 |
| C级 | 后期增强 | 调色、天气效果、UI叠加 | 低 |

### 标注格式

```markdown
[特效-A级] 陆霆琛从悬崖边跃起，身后金光大盛，
双翼展开跨度约5米，带着强烈的压迫感俯冲而下。
> 特效说明：金色能量翅膀，粒子效果，风压地面扬尘
> 预估时长：3-5秒
```

## 7. 分镜输出格式

### 单集分镜模板

```markdown
# EPXXX 分镜提示

## 基本信息
- **集数**: EPXXX
- **场景数**: X场
- **特效场景**: X场（等级明细）
- **预估拍摄量**: X个机位

---

## 场景 1

**景别**: [近景/中景/...]
**镜头运动**: [推/拉/固定/...]
**构图要点**: [构图描述]
**灯光氛围**: [灯光风格]
**表演重点**: [演员需要注意的情绪/动作]
**特效需求**: [如有]
**音效/音乐**: [声音设计提示]

---

## 场景 2
...
```

### 关键场景分镜示例

```markdown
## 场景 3 - 办公室对峙

**镜头1** | CU → 推 | 林晚推门的手
> 手指用力握住门把，指节发白。
> 🎵 低沉大提琴起

**镜头2** | FS | 门被推开，林晚全身入画
> 逆光，走廊的光打在她背后，脸在阴影中。

**镜头3** | OTS（过肩）| 从陆霆琛背后看向林晚
> 陆的肩膀占画面1/3，制造压迫感。

**镜头4** | MCU | 陆霆琛转身
> 侧光，半张脸在光中，半张在暗处。
> 表情从漫不经心到微微凝重。

**镜头5** | 正反打 | 两人对话
> 标准正反打，林晚用仰角（弱势），陆霆琛用俯角（强势）。
> 随着对话推进，角度逐渐平视（地位变化）。

**镜头6** | ECU | 林晚的眼神
> 特写她的眼睛，映出陆霆琛的倒影。
> 🎵 音乐骤停，3秒静音。

**镜头7** | CU | 辞职信拍在桌上
> 升格慢动作，纸张接触桌面的瞬间。
> 🔊 拍桌音效放大。
```

## 8. 分镜创作原则

1. **服务于情绪**: 镜头选择的第一标准是是否传达了正确的情绪
2. **经济原则**: 能用一个镜头表达的不用两个，短剧节奏不允许冗余
3. **可拍性优先**: 建议的镜头必须在实际拍摄条件下可实现
4. **竖屏克制**: 竖屏短剧尽量减少复杂运镜，以固定机位和简单推拉为主
5. **名场面重点**: 名场面/爆点的分镜要比日常场景更细致
6. **演员友好**: 分镜中标注表演重点，帮助演员理解情绪要求
