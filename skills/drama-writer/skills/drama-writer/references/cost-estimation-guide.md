# 成本预估方法论

本指南用于根据剧本内容生成制作成本预估报告。通过统计场景、角色、特效等维度，帮助制片团队评估预算和制作规模。

---

## 1. 统计维度

### 1.1 场景统计

从剧本中提取所有场景，按以下维度分类统计：

| 维度 | 分类 | 说明 |
|------|------|------|
| 场景类型 | 内景 / 外景 / 内外景 | 外景受天气和光线影响，成本波动大 |
| 场景复用 | 固定场景 / 一次性场景 | 固定场景搭建一次反复使用，性价比高 |
| 场景复杂度 | S / A / B / C | 见下方复杂度分级 |

#### 固定场景 vs 一次性场景

| 类型 | 说明 | 成本影响 |
|------|------|---------|
| 固定场景 | 反复出现的场景（主角家、办公室等） | 一次搭建/租赁，长期使用 |
| 一次性场景 | 只出现1-2次的场景 | 考虑是否值得搭建，能否用替代方案 |

### 1.2 角色统计

| 角色类型 | 定义 | 计费方式 |
|----------|------|---------|
| 主演 | 贯穿全剧的核心角色 | 按整部剧签约 |
| 重要配角 | 出现10集以上的配角 | 按天/按集 |
| 普通配角 | 出现3-10集的配角 | 按天 |
| 客串 | 出现1-2集的角色 | 按场次 |
| 群演 | 背景人物 | 按天按人头 |

### 1.3 特效统计

| 特效等级 | 说明 | 单次参考成本 |
|----------|------|-------------|
| S级 | 大型CG（仙术大战、城市级别） | ￥5,000-20,000/秒 |
| A级 | 中型特效（飞行、变身、能量波） | ￥1,000-5,000/秒 |
| B级 | 小型特效（光效、火焰、烟雾） | ￥200-1,000/秒 |
| C级 | 后期增强（调色、天气、UI） | ￥50-200/秒 |

> 以上为参考范围，实际成本因团队和地区差异很大。

## 2. 场景复杂度评级

### 评级标准

| 等级 | 场景描述 | 典型示例 |
|------|---------|---------|
| **S级** | 大型外景/复杂搭景/需大量群演 | 古装宫殿大殿、战场、大型宴会 |
| **A级** | 中型场景/需美术搭建/特殊道具 | 办公室全套、仙侠宗门殿堂、古风庭院 |
| **B级** | 普通内景/常见外景/简单布置 | 卧室、咖啡馆、公园、街道 |
| **C级** | 极简场景/纯背景/无需特殊布置 | 空房间对话、车内、电话场景 |

### 评级对成本的影响

| 等级 | 布景成本 | 拍摄时间 | 人员需求 |
|------|---------|---------|---------|
| S级 | 高 | 1-2天/场 | 全组 + 群演 |
| A级 | 中高 | 半天-1天/场 | 主要团队 |
| B级 | 低 | 2-4小时/场 | 核心团队 |
| C级 | 极低 | 1-2小时/场 | 最小团队 |

### 特效成本优化建议

1. **合并同类特效**: 相同类型的特效可以批量制作，降低单价
2. **用实拍替代**: 烟雾、火焰等能用实拍效果的优先实拍
3. **减少S级场景**: 控制S级特效场景在全剧3-5场以内
4. **集中安排**: 特效密集的集数集中拍摄，减少后期来回切换

## 6. 成本预估流程

### 执行步骤

1. **遍历所有已完成的剧本文件**，统计场景、角色、特效信息
2. **场景去重**，识别固定场景和一次性场景
3. **角色分级**，按出场集数和戏份量分类
4. **特效分级**，按描述判断特效等级
5. **服装道具提取**，从角色档案和剧本描述中汇总
6. **生成汇总表**，按上述模板输出
7. **标注风险点**，识别可能超预算的环节

### 注意事项

- 预估基于剧本内容，实际成本受地区、团队、档期等因素影响
- S级场景和大量外景是最大的成本变量
- 建议在大纲阶段就做初步预估，避免写出拍不起的剧本
- 成本预估是制片参考，不是精确报价
