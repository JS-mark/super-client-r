# 类型模板库

本文档收录短剧主流题材的创作模板。每个类型包含标准结构、角色原型、冲突模式和钩子套路，可直接套用或混搭。

**扩展方式**: 新增类型只需在文件末尾追加同格式的 section 即可。

---

## 1. 都市逆袭

### 类型特征

被低估的主角获得某种力量/身份/机遇，一步步打脸所有曾经看不起自己的人。核心卖点是"爽"——观众代入主角，享受逆袭快感。

### 标准剧情结构

| 阶段 | 集数范围（竖屏80集） | 核心事件 |
|------|---------------------|---------|
| 受辱期 | EP01-05 | 主角被各方羞辱、压迫，积累观众愤怒值 |
| 觉醒期 | EP06-10 | 获得逆袭资本（身份曝光/获得传承/贵人相助） |
| 小试牛刀 | EP11-25 | 逐一打脸小反派，展现实力，建立威信 |
| 中期对抗 | EP26-50 | 对抗中级反派，身份逐步揭秘，情感线推进 |
| 终极对决 | EP51-70 | 最大危机 + 最终反派，全面爆发 |
| 收官 | EP71-80 | 彻底逆袭，众人臣服，情感圆满 |

### 常见角色原型

| 角色 | 特征 | 台词风格 |
|------|------|---------|
| 废柴主角 | 初期隐忍，觉醒后果断霸气 | 前期沉默寡言，后期简短有力 |
| 势利眼 | 见风使舵，初期欺压后期跪舔 | 尖酸刻薄 → 谄媚讨好 |
| 隐藏大佬 | 暗中观察/保护主角 | 深沉、一针见血 |
| 前女友/前男友 | 抛弃主角，后悔莫及 | 高傲 → 酸涩 → 祈求 |
| 忠诚伙伴 | 唯一在低谷时帮助主角的人 | 直率、讲义气 |

### 典型场景设定

- 公司大厅（当众羞辱/当众打脸）
- 宴会/饭局（身份暴露名场面）
- 高端会所（实力碾压）
- 旧地重游（物是人非的反差）

### 常用冲突模式

1. **当众羞辱 → 当众打脸**: "你算什么东西？" → 身份揭晓，众人震惊
2. **被退婚 → 对方后悔**: 退婚时嘲笑 → 发现退掉的是最强者
3. **被看不起 → 实力碾压**: "就凭你？" → 一招制敌
4. **旧人重逢 → 地位逆转**: 曾经的欺压者变成求助者

### 钩子/反转套路

- **身份层层揭秘**: 每10集揭露一层身份（富二代 → 隐世家族继承人 → 终极身份）
- **以为到顶实则更高**: 打败一个强敌，发现背后还有更强的
- **被信任的人背叛**: 盟友/亲人反水，制造新危机
- **实力暴露时机**: 关键时刻不得不亮底牌

### 禁忌和注意事项

- 逆袭不能太快，前期受辱要足够让观众愤怒
- 打脸要有层次，不能每次都是同一套路
- 主角不能太圣母，该果断时必须果断
- 避免"我变强了但也变秃了"式的无意义牺牲

## 2. 都市甜宠

### 类型特征

男女主角从误会/契约/偶遇开始，经历心动、暧昧、阻碍，最终甜蜜HE。核心卖点是"甜"——高糖名场面密集，让观众嗑到上头。

### 标准剧情结构

| 阶段 | 集数范围（竖屏80集） | 核心事件 |
|------|---------------------|---------|
| 相遇期 | EP01-08 | 不打不相识/契约关系/误会开场 |
| 暧昧期 | EP09-25 | 日常相处产生心动，高糖互动密集 |
| 确认期 | EP26-35 | 表白/确认关系，甜蜜高峰 |
| 阻碍期 | EP36-55 | 第三者/家族反对/误会分手 |
| 虐心期 | EP56-65 | 分开/冷战/各自痛苦 |
| 破镜重圆 | EP66-80 | 化解误会，重新在一起，撒糖收官 |

### 常见角色原型

| 角色 | 特征 | 台词风格 |
|------|------|---------|
| 霸总 | 表面冷酷，对女主独宠，占有欲强 | 命令式 + 偶尔柔软反差 |
| 甜妹 | 元气、倔强、不畏权势 | 活泼直率，怼人可爱 |
| 白月光 | 男主前任或暗恋对象，温柔表面 | 温柔中带刺 |
| 绿茶闺蜜 | 假装好姐妹，暗中使绊子 | 表面关心实则阴阳 |
| 忠犬备胎 | 默默守护女主，不求回报 | 温柔隐忍 |

### 典型场景设定

- 咖啡馆/公司电梯（偶遇名场面）
- 男主豪宅（同居日常）
- 雨中/医院（心疼名场面）
- 宴会（身份曝光、吃醋场面）

### 常用冲突模式

1. **误会 → 吃醋 → 占有式表白**: 看到对方跟异性说话 → 霸气宣示主权
2. **嘴硬心软**: 嘴上说"跟我没关系" → 背后默默帮忙/心疼
3. **身份差异**: "我们不是一个世界的人" → 用行动证明真心
4. **契约到真心**: 假结婚/假恋爱 → 假戏真做

### 钩子/反转套路

- **甜蜜翻转**: 刚确认关系 → 发现重大秘密
- **第三者介入**: 前任回归/家族安排联姻
- **身份差距**: 突然发现对方的真实身份
- **善意谎言揭穿**: 为保护对方隐瞒 → 被误解为背叛

### 禁忌和注意事项

- 甜宠剧的虐心要适度，虐心期不能超过10集
- 男主霸道但不能真的蛮不讲理
- 第三者不能太无能（观众会觉得没威胁）
- 女主不能太傻白甜，要有自己的主见和底线
- 撒糖要有创意，不能都是壁咚和公主抱

## 3. 都市复仇

### 类型特征

主角曾遭受巨大伤害（被陷害/被灭门/被抛弃），重生或觉醒后展开精密复仇计划。核心卖点是"爽+谋略"——看主角步步为营，将仇人一个个拉下马。

### 标准剧情结构

| 阶段 | 集数范围（竖屏80集） | 核心事件 |
|------|---------------------|---------|
| 惨剧期 | EP01-05 | 展示主角被害的惨烈过程（闪回或现场） |
| 觉醒期 | EP06-10 | 重生/获得资源/决意复仇，制定计划 |
| 布局期 | EP11-30 | 步步为营，逐一收拾小反派，接近核心仇人 |
| 反击期 | EP31-55 | 与核心仇人正面交锋，形势反复 |
| 真相期 | EP56-70 | 揭露更深层的阴谋/真凶另有其人 |
| 终局 | EP71-80 | 终极复仇 + 情感归宿 |

### 常见角色原型

| 角色 | 特征 | 台词风格 |
|------|------|---------|
| 复仇者 | 表面温和，内心缜密，隐忍极强 | 平静中带杀意，偶尔爆发 |
| 核心仇人 | 表面成功人士，实则心狠手辣 | 虚伪客套 + 威胁恐吓 |
| 帮凶 | 当年参与伤害主角，现在心虚 | 色厉内荏 |
| 意外盟友 | 也被仇人伤害，与主角联手 | 直率、狠辣 |
| 无辜卷入者 | 与仇人有关但不知情 | 善良、纠结 |

### 常用冲突模式

1. **以其人之道还治其人之身**: 用当年害我的方式反过来对付你
2. **离间计**: 挑拨仇人阵营内部关系
3. **证据博弈**: 手握关键证据 → 被销毁 → 另有备份
4. **道德两难**: 复仇过程中伤及无辜，是否继续？

### 钩子/反转套路

- **仇人比想象中更强大**: 以为要赢了 → 仇人还有后手
- **盟友叛变**: 信任的人被策反或本就是内鬼
- **真相层层剥开**: 当年的真凶不是你以为的那个人
- **仇人有苦衷**: 发现仇人也是被迫的 → 真正的幕后黑手浮出水面

### 禁忌和注意事项

- 复仇计划要有逻辑，不能全靠巧合
- 主角复仇手段不能太毒辣（会失去观众同情）
- 仇人不能太蠢，否则复仇没有成就感
- 适度加入情感线，纯复仇会让观众疲劳
- 复仇动机必须让观众认同——"活该被报复"

## 4. 仙侠

### 类型特征

凡人踏上修仙之路，从废材到强者，历经修炼、斗法、宗门争斗、界域冒险。核心卖点是"热血+升级"——修炼体系提供持续的升级快感。

### 标准剧情结构

| 阶段 | 集数范围（竖屏80集） | 核心事件 |
|------|---------------------|---------|
| 凡人期 | EP01-05 | 主角为废材，被欺辱，但有不屈之心 |
| 奇遇期 | EP06-12 | 获得仙缘/传承/神器，开始修炼 |
| 崭露头角 | EP13-30 | 在宗门/门派中崛起，击败同辈 |
| 宗门大比 | EP31-45 | 大型比武/试炼，名声大噪 |
| 阴谋浮现 | EP46-60 | 卷入上层势力争斗，生死危机 |
| 突破飞升 | EP61-75 | 突破关键境界，正面对抗大反派 |
| 终极之战 | EP76-80 | 最终对决 + 得道/归隐 |

### 修炼体系（标准模板，可自定义）

```
炼气 → 筑基 → 金丹 → 元婴 → 化神 → 渡劫 → 飞升
```

每个大境界分为初期、中期、后期、圆满四个小阶段。

### 常见角色原型

| 角色 | 特征 | 台词风格 |
|------|------|---------|
| 废材主角 | 资质差但意志坚定，有隐藏体质/血脉 | 初期谦逊，后期霸气 |
| 老爷爷/残魂 | 寄宿在神器中的强者残魂 | 老辣、调侃、傲然 |
| 天才师兄/师姐 | 正道天才，前期看不起主角 | 高傲 → 服气 |
| 魔道反派 | 为达目的不择手段 | 阴鸷、狂妄 |
| 红颜知己 | 仙门弟子，与主角并肩作战 | 温婉或飒爽 |

### 典型场景设定

- 仙门/宗门大殿
- 密境/秘境/试炼之地
- 悬崖/瀑布（修炼、突破）
- 坊市/拍卖会（交易、冲突）
- 战场（宗门大战、魔修入侵）

### 常用冲突模式

1. **以弱胜强**: 低境界击败高境界，越级战斗
2. **修炼资源争夺**: 天材地宝引发争斗
3. **宗门考核淘汰**: 限时任务、排名竞争
4. **身世之谜**: 逐步揭露主角的隐藏血脉/身世

### 钩子/反转套路

- **突破关键时刻被打断**: 即将突破 → 强敌来袭
- **隐藏实力暴露**: 一直藏拙 → 关键时刻暴走
- **宝物认主**: 所有人抢的宝物，只认主角
- **灭门之仇**: 发现宗门/家族灭门真相

### 禁忌和注意事项

- 修炼体系要自洽，不能随意打破已有设定
- 升级节奏要合理，不能一集突破三个境界
- 法术对决要有策略感，不能只是"比谁能量大"
- 仙侠世界的价值观要统一——丛林法则 or 正义守护
- 短剧仙侠要简化世界观，不要铺太大的设定

## 5. 古装宫斗

### 类型特征

女主进入后宫/王府，在嫔妃/侧室之间的勾心斗角中步步为营，最终登上高位。核心卖点是"智斗+权谋"——看女主用智慧在后宫生存、上位。

### 标准剧情结构

| 阶段 | 集数范围（竖屏80集） | 核心事件 |
|------|---------------------|---------|
| 入宫期 | EP01-08 | 选秀入宫/被迫入府，初识规矩 |
| 立足期 | EP09-20 | 小心谨慎，结交盟友，躲过初期陷害 |
| 崛起期 | EP21-40 | 开始反击，扳倒第一个对手 |
| 争宠期 | EP41-55 | 得宠引妒，多方围攻 |
| 至暗时刻 | EP56-65 | 被陷害至最低谷，失宠/打入冷宫 |
| 翻盘 | EP66-75 | 绝境翻盘，揭露最大对手真面目 |
| 登顶 | EP76-80 | 最终上位/与心爱之人在一起 |

### 后宫等级体系（标准模板）

```
皇后 → 贵妃 → 妃 → 嫔 → 贵人 → 常在 → 答应
```

### 常见角色原型

| 角色 | 特征 | 台词风格 |
|------|------|---------|
| 聪慧女主 | 表面柔弱，内心睿智 | 温和中暗藏机锋 |
| 跋扈宠妃 | 仗势欺人，手段毒辣 | 尖酸傲慢 |
| 心机女配 | 笑里藏刀，善于伪装 | 温柔体贴的假面 |
| 忠心侍女 | 主角最信任的人 | 忠诚、机灵 |
| 帝王 | 多疑、平衡各方势力 | 威严、深不可测 |
| 太后 | 掌握最终权力的隐藏大佬 | 慈祥或威严 |

### 常用冲突模式

1. **嫁祸**: 在女主宫中放毒/放证据，栽赃陷害
2. **争宠**: 利用帝王面前的表现争夺恩宠
3. **联盟与背叛**: 暂时结盟 → 利益冲突后背叛
4. **子嗣争夺**: 怀孕/孩子成为最大的筹码和目标

### 钩子/反转套路

- **被信任的人背叛**: 忠心侍女/盟友被策反
- **翻案逆转**: 被冤枉 → 找到证据当众翻案
- **帝王心思**: 以为失宠 → 其实是帝王在暗中保护
- **身世揭秘**: 女主/女配的真实身份

### 禁忌和注意事项

- 宫斗手段要符合时代背景，不能太现代思维
- 女主要有底线，不能为上位不择手段（失去观众共鸣）
- 帝王不能太蠢——否则宫斗毫无意义
- 注意古装礼仪用语的准确性
- 服装描写要有时代感，不能混搭

## 6. 古装权谋

### 类型特征

围绕皇位争夺、权力博弈、国家存亡展开的政治斗争剧。核心卖点是"权谋+格局"——庙堂之上的纵横捭阖，比后宫更宏大的博弈。

### 标准剧情结构

| 阶段 | 集数范围（横屏30集） | 核心事件 |
|------|---------------------|---------|
| 棋局初开 | EP01-05 | 展示权力格局，主角的处境和目标 |
| 暗流涌动 | EP06-12 | 各方势力暗中布局，主角寻找盟友 |
| 初次交锋 | EP13-18 | 第一次正面对抗，小胜或惨胜 |
| 局势逆转 | EP19-22 | 最大的危机，盟友倒戈或计划泄露 |
| 绝地反击 | EP23-27 | 置之死地而后生，终极计划启动 |
| 定鼎天下 | EP28-30 | 最终胜利 + 权力代价 |

### 常见角色原型

| 角色 | 特征 | 台词风格 |
|------|------|---------|
| 隐忍皇子 | 韬光养晦，深藏不露 | 平淡克制，关键时刻锋芒毕露 |
| 权臣 | 把持朝政，老谋深算 | 阴阳怪气，话中有话 |
| 谋士 | 智计百出，忠于主角 | 引经据典，冷静分析 |
| 将军 | 战场上的关键力量 | 直爽豪迈 |
| 昏君/暴君 | 多疑、暴虐、被操控 | 喜怒无常 |

### 常用冲突模式

1. **朝堂辩论**: 以言论为武器，当众驳倒对手
2. **情报战**: 安插眼线、截获密信
3. **军事政变**: 兵权争夺、宫变
4. **联姻博弈**: 政治婚姻中的利益交换

### 钩子/反转套路

- **一石二鸟**: 看似针对A的计策，真正目标是B
- **将计就计**: 明知是陷阱，故意入局反杀
- **死间计**: 牺牲自己人来取信于敌
- **退位让贤**: 看似放弃权力 → 实则更大的布局

### 禁忌和注意事项

- 权谋剧智商要在线，不能有太多低级错误
- 各方势力要平衡，不能一方独大到没悬念
- 历史背景可架空，但政治逻辑要真实
- 短剧权谋要简化线索，不超过3条主线同时推进
- 建议用横屏格式，适合大场面和群像戏

## 7. 悬疑

### 类型特征

围绕一个核心谜团展开，通过线索收集、推理分析逐步揭露真相。核心卖点是"烧脑+反转"——让观众跟着猜真凶/真相，被反转震撼。

### 标准剧情结构

| 阶段 | 集数范围（横屏30集） | 核心事件 |
|------|---------------------|---------|
| 事件发生 | EP01-03 | 核心事件（案件/失踪/异常）发生 |
| 初步调查 | EP04-10 | 收集线索，建立嫌疑人列表 |
| 深入迷局 | EP11-18 | 更多线索出现，误导和真线索混杂 |
| 危机升级 | EP19-23 | 调查者陷入危险/第二起事件 |
| 真相浮现 | EP24-27 | 关键线索串联，真相方向明确 |
| 终极揭秘 | EP28-30 | 真相大白 + 最终反转 |

### 常见角色原型

| 角色 | 特征 | 台词风格 |
|------|------|---------|
| 侦探/调查者 | 观察力强，执着追真相 | 冷静、逻辑性强 |
| 核心嫌疑人 | 有嫌疑但不是真凶 | 躲闪、矛盾 |
| 真凶 | 隐藏最深的人 | 正常人伪装 |
| 受害者关联人 | 提供关键信息 | 情绪化、有所隐瞒 |
| 搅局者 | 出于私人目的干扰调查 | 挑衅、转移注意 |

### 叙事技巧

- **多线叙事**: 2-3条调查线并行，最终交汇
- **不可靠叙述**: 某个角色的叙述是假的
- **时间线错乱**: 非线性叙事，制造信息差
- **密室/封闭环境**: 限定空间增加紧张感

### 常用冲突模式

1. **证据矛盾**: 两条线索互相矛盾，必须找出哪条是假的
2. **嫌疑人陆续排除**: 每排除一个 → 接近真凶
3. **调查者被威胁**: 接近真相 → 收到威胁/暗杀
4. **同伙反水**: 知情者良心发现，提供关键证词

### 钩子/反转套路

- **不在场证明崩塌**: 看似铁证 → 发现漏洞
- **真凶就在身边**: 最信任的人就是凶手
- **受害者其实没死**: 或者受害者本身就是阴谋的一部分
- **连环反转**: 你以为A是凶手 → 其实是B → 最终竟然是C

### 禁忌和注意事项

- 线索必须公平——揭秘时用的证据，之前都要出现过
- 不能用"凶手是新角色"来制造反转（观众会觉得被骗）
- 逻辑必须自洽，经不起推敲的悬疑不如不写
- 节奏要紧凑，每集必须有新线索或新发现
- 建议用横屏格式，适合复杂叙事

## 8. 穿越重生

### 类型特征

主角从现代穿越到古代/异世界，或死后重回过去某个时间节点。利用前世记忆/现代知识改变命运。核心卖点是"先知金手指+命运改写"。

### 标准剧情结构

| 阶段 | 集数范围（竖屏80集） | 核心事件 |
|------|---------------------|---------|
| 穿越/重生 | EP01-03 | 穿越或重生发生，主角确认处境 |
| 适应期 | EP04-10 | 利用先知避开前世的第一个坑 |
| 蝴蝶效应 | EP11-30 | 改变命运引发连锁反应，新变量出现 |
| 命运反扑 | EP31-50 | 某些命运无法避免，或出现前世没有的新危机 |
| 终极选择 | EP51-70 | 面对最大的命运节点，必须做出选择 |
| 命运改写 | EP71-80 | 成功改写命运 / 接受新的人生 |

### 常见角色原型

| 角色 | 特征 | 台词风格 |
|------|------|---------|
| 穿越/重生者 | 有前世记忆，看似先知 | 偶尔说出超越时代的话 |
| 前世仇人 | 上辈子害过主角的人 | 还没变坏或已经在布局 |
| 前世恩人 | 上辈子帮过主角的人 | 善良，值得回报 |
| 命运变数 | 前世不存在的新角色 | 不按剧本走的人 |
| 另一个穿越者 | 同样穿越过来的对手 | 知道未来的同类 |

### 常用冲突模式

1. **先知悖论**: 因为知道未来而改变了未来 → 先知不再灵验
2. **蝴蝶效应失控**: 改变一件小事 → 引发意想不到的大事
3. **同类对抗**: 遇到另一个穿越者/重生者
4. **不可改变的命运**: 某些事无论怎么做都会发生

### 钩子/反转套路

- **前世记忆出错**: 以为知道未来 → 发现记忆和现实不符
- **改变了不该改变的**: 救了一个人 → 导致更多人遭殃
- **身份暴露**: 被人发现主角知道不该知道的事
- **时间线分歧**: 这已经不是前世的那个世界了

### 禁忌和注意事项

- 先知能力要有限制，不能什么都知道（否则无悬念）
- 改变命运要有代价，不能太轻松
- 不能把所有问题都用"我知道未来"来解决
- 主角要有成长，不能只靠前世经验躺赢
- 重生/穿越的原因最好有交代（即使是模糊的）
