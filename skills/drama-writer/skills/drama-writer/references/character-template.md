本模板用于设计短剧角色。短剧角色的核心要求：**辨识度高、记忆点强、出场即有标签**。

## 2. 短剧特有设计

### 出场记忆点设计

短剧角色的第一次出场至关重要——观众在3秒内决定是否记住这个角色。

| 要素 | 设计 |
|------|------|
| **出场集数** | EP__ |
| **出场方式** | [动作出场/台词出场/反差出场/悬念出场] |
| **第一句台词** | 必须有记忆点 |
| **第一个动作** | 必须体现性格 |
| **观众第一印象** | 想让观众觉得这个人是__的 |

**出场方式类型**:

| 类型 | 说明 | 示例 |
|------|------|------|
| 动作出场 | 用一个动作定义角色 | 女主一脚踹开门，扔出辞职信 |
| 台词出场 | 第一句话就刻入记忆 | "三年了，够了。" |
| 反差出场 | 跟观众预期完全相反 | 所有人以为的废物 → 一出手就制服歹徒 |
| 悬念出场 | 先不露脸，只给背影/声音 | 暗处传来低沉的声音："让她进来。" |
| 关联出场 | 通过另一个角色的反应引出 | 所有人听到名字脸色大变 |

## 3. 角色塑造技巧

### 3.1 标签化入场

短剧没有时间慢慢铺垫角色。每个角色出场必须带标签——用一个细节、一句话、一个动作让观众立刻记住。

**❌ 错误**:
> 李明是一个性格温和但内心坚定的人，平时话不多，但关键时刻总能靠得住。

**✅ 正确**:
> 李明推门进来，西装革履，但左手无名指上缠着一圈旧创可贴——那是他养的流浪猫抓的。

### 3.2 台词即性格

不同角色说同一件事，方式必须不同。如果互换台词也不违和，说明角色不够鲜明。

**同一件事的不同说法**（"你被解雇了"）:

| 角色类型 | 说法 |
|----------|------|
| 霸总 | "从明天起，你不用来了。" |
| 心机女 | "姐姐帮你跟老板说了，让你好好休息一段时间。" |
| 莽夫 | "老子炒你鱿鱼！滚！" |
| 阴险小人 | "公司最近在优化人员结构……你懂的。" |

### 3.3 行为定义人物

不要用形容词告诉观众角色是什么样的人——用行为展示。

**❌ 告诉**: "她是一个善良的人。"

**✅ 展示**: 她把唯一的伞递给了对面的孕妇，自己淋着雨跑进地铁站。

### 3.4 反差制造魅力

角色魅力的最大来源是反差——逻辑上矛盾但情感上成立的特质。

| 反差类型 | 示例 |
|----------|------|
| 身份反差 | 白天是公司保安，晚上是地下拳场冠军 |
| 行为反差 | 对所有人冷酷无情，但对一只猫温柔至极 |
| 言行反差 | 嘴上说"跟我没关系"，身体却冲上去挡刀 |
| 过去/现在反差 | 曾经的天之骄子，现在扫大街 |

### 3.5 对手必须强大

反派/对手的强度决定了主角胜利的含金量。弱鸡反派 = 无趣的胜利。

**反派设计三要素**:
1. **有动机**: 不是为了坏而坏，要有自己的逻辑
2. **有能力**: 能真正威胁到主角，而不是纸老虎
3. **有底线**（可选但加分）: 有底线的反派比纯粹邪恶更有张力

### 3.6 配角不是工具人

即使是配角，也要有自己的欲望和行为逻辑，而不是纯粹为主角服务。

**❌ 工具人**: 闺蜜永远在主角需要时出现，没有自己的生活

**✅ 有灵魂的配角**: 闺蜜有自己的感情烦恼，有时候帮主角出主意的同时也在处理自己的问题

## 5. 角色设计清单

创建每个角色时，确认以下要点：

- [ ] 有2-3个视觉标志（不看名字也能认出来）
- [ ] 台词风格独特（互换台词会违和）
- [ ] 有至少一组性格反差
- [ ] 出场方式有记忆点
- [ ] 有清晰的角色弧线
- [ ] 预设了2-3句名场面台词
- [ ] 人物关系中有至少一组动态变化
- [ ] 服装造型随剧情阶段变化
