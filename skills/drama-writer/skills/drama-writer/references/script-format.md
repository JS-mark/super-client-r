# 剧本格式规范

本文档定义短剧剧本的标准书写格式。所有剧本输出必须遵循此规范，确保格式统一、可读性强、可直接用于拍摄。

---

## 1. 单集剧本结构

每集剧本文件的标准结构如下：

```markdown
# EPXXX - 集标题

> **所属项目**: 剧名
> **集数**: 第X集 / 共X集
> **预估时长**: X分钟
> **台词字数**: X字
> **场景数**: X场

---

[场景内容]

---

🪝 **EP HOOK**: [本集结尾钩子描述]

---

> **下集预告**: [1-2句下集核心冲突提示]
```

---

## 2. 场景标题

每个场景以场景标题开头，格式为：

```
**场景 X. 内景/外景 地点 - 时间**
```

**规则**：
- 场景编号从 1 开始，按出场顺序递增
- 内景/外景必须标注（内景、外景、内外景）
- 地点具体明确（"总裁办公室"而非"办公室"）
- 时间标注：白天、夜晚、黄昏、清晨、深夜

**示例**：

```
**场景 1. 内景 陆氏集团·总裁办公室 - 白天**
```

```
**场景 5. 外景 城中村·巷口 - 夜晚**
```

```
**场景 12. 内外景 医院·走廊/病房 - 清晨**
```

## 3. 动作描述

动作描述使用引用块（blockquote）格式，现在时态，简洁有力。

**规则**：
- 使用现在时态（"推门而入"而非"推门走了进去"）
- 简短有力，每段不超过3行
- 描写必须可视化——能直接转化为镜头画面
- 角色首次出场需简要外貌描写（2-3个标志特征）

**格式**：

```markdown
> 林晚推门而入，职业装笔挺，手里攥着一份辞职信。
> 她扫了一眼空荡荡的办公桌，嘴角微微上扬。
```

**角色首次出场示例**：

```markdown
> 陆霆琛（28岁，棱角分明的侧脸，眼神冷淡如冰）坐在落地窗前，
> 手指漫不经心地转着钢笔。
```

## 4. 角色台词

角色台词是剧本的核心。格式要求严格统一。

### 基本格式

```markdown
**角色名**
台词内容

**角色名**（表演指示）
台词内容
```

### 表演指示

用括号标注角色说台词时的状态、动作、情绪，紧跟在角色名后面。

```markdown
**林晚**（冷笑）
"你以为辞退我，就能掩盖真相？"

**陆霆琛**（放下钢笔，眼神微沉）
"林小姐，我给你三秒钟，收回这句话。"

**林晚**（直视他的眼睛，一字一顿）
"一、二、三。时间到了，陆总。"
```

### 台词长度要求

| 平台类型 | 单句台词上限 | 单段台词上限 |
|----------|-------------|-------------|
| 竖屏短剧 | 15字 | 3句 |
| 横屏微短剧 | 25字 | 5句 |

### 旁白/画外音

```markdown
**旁白（VO）**
三年前的林晚不会想到，她的人生即将被彻底改写。
```

### 电话/对讲/短信

```markdown
**陆霆琛**（电话中，声音冰冷）
"把林晚的资料给我调出来，所有的。"

> 手机屏幕显示一条短信：
> **「你确定要跟陆家作对？」**
```

## 5. 镜头提示

镜头提示为可选内容，用方括号标注，辅助导演理解编剧意图。

**格式**：

```markdown
[特写：林晚握紧的拳头，指节发白]
[推镜头：从全景推到陆霆琛惊愕的脸]
[俯拍：两人在空旷大厅对峙，画面形成压迫感]
[手持跟拍：林晚快步穿过走廊]
[升格慢动作：戒指从手中滑落]
```

**常用镜头标注**：

| 镜头类型 | 缩写 | 用途 |
|----------|------|------|
| 特写 | CU | 情绪细节、关键道具 |
| 近景 | MS | 对话、表情 |
| 中景 | MLS | 两人互动 |
| 全景 | FS | 环境交代、群戏 |
| 远景 | LS | 场景建立、氛围 |
| 过肩 | OTS | 对话正反打 |

---

## 6. 音效与音乐提示

用尖括号标注，置于相关动作或台词前后。

```markdown
<BGM：紧张悬疑音乐渐起>

> 林晚缓缓打开文件夹。

<音效：翻页声>

**林晚**（声音颤抖）
"这不可能……"

<BGM：音乐骤停>
<音效：手机震动>
```

**常用标注类型**：
- `<BGM：描述>` — 背景音乐
- `<音效：描述>` — 环境音效或动作音效
- `<音乐：停>` — 音乐停止
- `<静音>` — 完全安静，营造压迫感

---

## 7. 转场

场景之间的转场用分隔线 + 转场类型标注。

```markdown
---
**【转场：硬切】**
---
```

**常用转场类型**：

| 转场 | 用途 | 示例场景 |
|------|------|---------|
| 硬切 | 快节奏、冲突对比 | 吵架 → 独自哭泣 |
| 闪回 | 回忆、往事 | 当前场景 → 三年前 |
| 闪白 | 强烈冲击、觉醒 | 被打一拳 → 白光 → 倒地 |
| 闪黑 | 失去意识、时间跳跃 | 昏倒 → 黑屏 → 醒来 |
| 叠化 | 抒情、时间流逝 | 回忆画面叠到现在 |
| 匹配剪辑 | 画面相似元素衔接 | 合上的眼 → 打开的门 |

---

## 8. 集末钩子标记

每集结尾必须有钩子。用特殊格式标注，确保不遗漏。

```markdown
🪝 **EP HOOK**: 门外传来敲门声，林晚脸色骤变——来的人，不该出现在这里。
```

**钩子类型参考**：

| 类型 | 说明 | 示例 |
|------|------|------|
| 悬念钩子 | 留下未解之谜 | "门被推开，站在门口的人让所有人都愣住了。" |
| 反转钩子 | 颠覆已知信息 | "她低头看向验孕棒——两条线。" |
| 危机钩子 | 角色陷入险境 | "电话那头的声音让她血液凝固：'你女儿在我手上。'" |
| 情感钩子 | 情感冲击 | "他终于说出了那句话：'我从来没有爱过你。'" |
| 信息钩子 | 揭露关键信息 | "遗嘱上的名字，不是她的——而是她最信任的那个人。" |
| 行动钩子 | 角色做出重大决定 | "她拿起手机，拨通了那个三年没打过的号码。" |

---

## 9. 特殊标记汇总

| 标记 | 格式 | 用途 |
|------|------|------|
| 场景标题 | `**场景 X. 内/外景 地点 - 时间**` | 场景开始 |
| 动作描述 | `> 描述文字` | 可视化动作/环境 |
| 角色台词 | `**角色名**（指示）\n台词` | 对话 |
| 镜头提示 | `[镜头类型：描述]` | 导演参考 |
| 音效音乐 | `<BGM/音效：描述>` | 声音设计参考 |
| 转场 | `**【转场：类型】**` | 场景衔接 |
| 集末钩子 | `🪝 **EP HOOK**: 描述` | 每集结尾悬念 |
| 旁白 | `**旁白（VO）**` | 画外音 |
| 短信/文字 | `**「内容」**` | 屏幕文字 |

## 10. 完整单场示例

```markdown
**场景 3. 内景 陆氏集团·总裁办公室 - 白天**

<BGM：低沉的大提琴，暗流涌动>

> 宽敞的办公室，落地窗外是整个城市的天际线。
> 陆霆琛站在窗前，背对着门口，手里握着一份文件。

> 门被推开。林晚走进来，脚步声在大理石地面上清脆作响。
> 她穿着洗得发白的职业装，但腰板挺得笔直。

**陆霆琛**（没有回头）
"辞职信？"

**林晚**
"对。即日生效。"

[特写：陆霆琛转动钢笔的手指顿了一下]

> 陆霆琛缓缓转身。他看向林晚的眼神像是在审视一件不太重要的物品。

**陆霆琛**（把文件扔到桌上）
"给你三天时间，把Q3的审计报告交了再走。"

**林晚**（上前一步，把辞职信拍在桌上）
"陆总，审计报告的问题，您比我清楚。"

<BGM：音乐骤停>
<音效：三秒钟的完全安静>

> 空气凝固。陆霆琛的眼神终于有了温度——冰冷的怒意。

**陆霆琛**（压低声音，一字一顿）
"你在威胁我？"

**林晚**（直视他，毫不退让）
"我只是在陈述事实。"

[推镜头：从两人对峙的全景推到林晚坚定的眼神]

> 她转身走向门口，步伐从容。

**林晚**（在门口停下，没有回头）
"陆总，有些秘密，捂不住的。"

> 门轻轻合上。

<音效：门锁咔哒声>

> 陆霆琛站在原地，面部肌肉几不可见地抽动了一下。
> 他拿起手机，拨出一个号码。

**陆霆琛**（声音冰冷）
"查她。查干净。"
```
