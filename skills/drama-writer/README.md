# Drama Writer Plugin (短剧剧本创作助手)

短剧剧本创作插件，支持竖屏短剧和横屏微短剧的全流程创作。

## 功能概览

- **竖屏短剧**: 1-3分钟/集，60-100集，适合抖音/快手/微信视频号
- **横屏微短剧**: 5-15分钟/集，20-40集，适合YouTube/长视频平台
- **8大题材模板**: 都市逆袭、都市甜宠、都市复仇、仙侠、古装宫斗、古装权谋、悬疑、穿越重生
- **全流程覆盖**: 项目设置 → 故事大纲 → 角色设计 → 分集大纲 → 剧本创作 → 分镜提示 → 成本预估

## 安装

将 `skills/drama-writer/` 目录放置到 Super Client R 的 skills 目录下即可。插件会自动被识别和加载。

## 使用方式

### 快速开始

使用 `/drama` 命令启动创作：

```
/drama                    # 使用默认路径 ~/Desktop/drama/
/drama ~/my-project       # 指定自定义路径
```

### 自然语言触发

以下关键词会自动触发本技能：

- "写短剧"、"写剧本"、"短剧创作"
- "分集大纲"、"下一集"、"继续写剧本"
- "写分镜"、"角色设计"、"成本预估"

### 创作流程

1. **项目设置** — 选择平台（竖屏/横屏）、题材、集数
2. **故事大纲** — 高概念、主线剧情、分幕规划、反转节点
3. **角色设计** — 角色档案、关系网、台词风格、出场记忆点
4. **分集大纲** — 每集核心冲突和结尾钩子
5. **剧本创作** — 按专业格式输出完整剧本
6. **分镜提示**（可选）— 镜头、构图、转场建议
7. **成本预估**（可选）— 场景、角色、特效统计和预算参考

### 续写已有项目

再次使用 `/drama` 命令，插件会自动加载已有项目的进度，从上次中断的地方继续。

## 文件结构

### 插件文件

```
skills/drama-writer/
├── .claude-plugin/
│   └── plugin.json                  # 插件元数据
├── commands/
│   └── drama.md                     # /drama 斜杠命令
├── skills/
│   └── drama-writer/
│       ├── SKILL.md                 # 核心技能定义
│       └── references/
│           ├── script-format.md     # 剧本格式规范
│           ├── genre-templates.md   # 类型模板库（8大题材）
│           ├── outline-template.md  # 短剧大纲模板
│           ├── character-template.md # 角色档案模板
│           ├── storyboard-guide.md  # 分镜场景指南
│           └── cost-estimation-guide.md # 成本预估方法论
└── README.md                        # 本文件
```

### 项目输出文件

```
<DRAMA_ROOT>/                        # 默认 ~/Desktop/drama/
├── project.md                       # 项目配置
├── outline.md                       # 故事大纲 + 分集大纲
├── characters.md                    # 角色档案
├── progress.md                      # 创作进度 + 伏笔管理
├── cost-estimate.md                 # 成本预估报告
├── episodes/
│   ├── EP001-标题.md                # 单集剧本
│   └── ...
└── storyboard/
    ├── EP001-分镜.md                # 单集分镜
    └── ...
```

## 题材支持

| 题材 | 推荐平台 | 核心卖点 |
|------|---------|---------|
| 都市逆袭 | 竖屏 | 爽感、打脸、逆袭 |
| 都市甜宠 | 竖屏 | 高糖、嗑CP、心动 |
| 都市复仇 | 竖屏 | 谋略、爽感、解气 |
| 仙侠 | 竖屏/横屏 | 热血、升级、斗法 |
| 古装宫斗 | 竖屏/横屏 | 智斗、权谋、逆袭 |
| 古装权谋 | 横屏 | 格局、博弈、烧脑 |
| 悬疑 | 横屏 | 烧脑、反转、推理 |
| 穿越重生 | 竖屏 | 先知、改命、爽感 |

## 扩展题材

`references/genre-templates.md` 支持扩展，在文件末尾按标准格式追加新题材 section 即可。
