# 自动更新

Super Client R 内置自动更新功能，确保你始终使用最新版本。

## 更新流程

### 1. 检查更新

应用启动时会自动在后台检查是否有新版本可用。你也可以手动检查：

- 进入 **设置** > **关于**
- 点击 **检查更新** 按钮

### 2. 下载更新

当检测到新版本时：

- 应用会通知你有新版本可用
- 显示新版本的版本号
- 点击 **下载更新** 开始下载
- 下载过程中会显示进度百分比

### 3. 安装更新

下载完成后：

- 会收到「更新已准备就绪」的通知
- 可以选择 **立即安装** 重启应用
- 或在下次关闭应用时自动安装

## 更新状态

更新过程中会经历以下状态：

| 状态 | 说明 |
|------|------|
| 检查中 (Checking) | 正在检查新版本 |
| 有新版本 (Available) | 发现新版本可用 |
| 已是最新 (Not Available) | 当前已是最新版本 |
| 下载中 (Progress) | 正在下载更新包 |
| 已下载 (Downloaded) | 更新包下载完成 |
| 错误 (Error) | 检查或下载过程出错 |

## 更新设置

- **自动下载**：默认关闭，开启后检测到新版本时会自动下载
- **退出时自动安装**：默认开启，关闭应用时自动安装已下载的更新

## 跨平台支持

自动更新功能基于 electron-updater 实现，支持 macOS、Windows 和 Linux 平台。

## 手动更新

如果自动更新出现问题，可以手动下载最新版本安装包进行更新。
