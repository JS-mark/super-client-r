# 工作区

工作区功能帮助你组织和隔离不同场景的对话和设置，让 AI 使用更加高效和有条理。

## 工作区类型

Super Client R 提供 4 种工作区类型：

| 类型 | 说明 | 适用场景 |
|------|------|---------|
| 个人 (Personal) | 个人日常使用 | 日常问答、学习笔记 |
| 工作 (Work) | 工作相关 | 代码开发、文档写作 |
| 项目 (Project) | 特定项目 | 项目管理、方案讨论 |
| 临时 (Temp) | 临时使用 | 一次性任务、快速测试 |

## 创建工作区

1. 进入 **工作区** 页面
2. 点击 **新建工作区** 按钮
3. 填写工作区名称和描述
4. 选择工作区类型
5. 选择工作区颜色（10 种预设颜色可选）
6. 配置工作区设置
7. 保存

## 工作区设置

每个工作区可以独立配置以下参数：

- **默认模型** - 设置该工作区使用的 AI 模型
- **系统提示词** - 自定义 AI 的角色和行为
- **温度 (Temperature)** - 调节回复的随机性（默认 0.7）
- **最大上下文长度** - 设置对话的上下文窗口大小（默认 4000）
- **自动保存** - 是否自动保存对话记录
- **自定义变量** - 存储工作区专属的配置参数

## 管理工作区

### 切换工作区

点击工作区卡片即可切换到对应工作区，所有后续对话将在该工作区中进行。

### 搜索工作区

使用搜索栏按名称或描述快速查找工作区。

### 复制工作区

可以复制现有工作区及其所有设置，快速创建类似配置的新工作区。

### 设为默认

将常用的工作区设为默认，应用启动时自动使用该工作区。

### 导入/导出

- **导出**：将工作区配置导出为 JSON 文件
- **导入**：从 JSON 文件导入工作区配置

### 删除工作区

可以删除不再需要的工作区，但默认工作区不可删除。

## 工作区统计

工作区页面提供概览统计：

- 工作区总数
- 各类型工作区数量
- 每个工作区的会话数量和消息数量
- 创建时间
