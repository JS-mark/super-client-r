# 快捷键

Super Client R 提供丰富的键盘快捷键，提升操作效率。所有快捷键均可自定义。

## 默认快捷键

::: tip 说明
下表中 `Mod` 表示修饰键：macOS 上为 `⌘ Command`，Windows/Linux 上为 `Ctrl`。
:::

### 全局快捷键

| 功能 | macOS | Windows/Linux |
|------|-------|---------------|
| 新建对话 | `⌘ N` | `Ctrl+N` |
| 快速搜索 | `⌘ K` | `Ctrl+K` |
| 切换侧边栏 | `⌘ B` | `Ctrl+B` |
| 打开设置 | `⌘ ,` | `Ctrl+,` |
| 打开收藏 | `⌘ ⇧ B` | `Ctrl+Shift+B` |

### 聊天快捷键

| 功能 | macOS | Windows/Linux |
|------|-------|---------------|
| 发送消息 | `↵ Enter` | `Enter` |
| 换行 | `⇧ ↵` | `Shift+Enter` |
| 清除对话 | `⌘ ⇧ K` | `Ctrl+Shift+K` |
| 聚焦输入框 | `⌘ I` | `Ctrl+I` |

### 导航快捷键

| 功能 | macOS | Windows/Linux |
|------|-------|---------------|
| 跳转到聊天 | `⌘ 1` | `Ctrl+1` |
| 跳转到模型 | `⌘ 2` | `Ctrl+2` |
| 跳转到技能 | `⌘ 3` | `Ctrl+3` |
| 跳转到插件 | `⌘ 4` | `Ctrl+4` |
| 跳转到收藏 | `⌘ 5` | `Ctrl+5` |

## 自定义快捷键

### 修改快捷键

1. 进入 **设置** > **快捷键**
2. 找到要修改的快捷键
3. 点击快捷键按钮进入录制模式
4. 按下新的快捷键组合
5. 自动保存

### 冲突检测

修改快捷键时，系统会自动检测是否与已有快捷键冲突：

- 如果检测到冲突，会提示冲突的快捷键信息
- 需要解决冲突后才能保存

### 启用/停用

- 可以全局启用或停用快捷键功能
- 也可以单独停用某个快捷键

### 恢复默认

点击 **恢复默认** 按钮可以将所有快捷键重置为默认配置。

## 平台差异

快捷键显示会根据当前平台自动调整：

**macOS 符号**：`⌘` `⌥` `⇧` `↵` `⌫` `⇥` `↑` `↓` `←` `→`

**Windows 显示**：`Ctrl` `Alt` `Shift` `Enter` `Delete` `Tab` 等文本形式
