# 悬浮窗

Super Client R 提供悬浮窗（Float Widget）功能，以小型浮动窗口的形式常驻桌面，方便快速访问 AI 功能。

## 功能概述

悬浮窗是一个始终置顶的胶囊式小窗口，让你无需切换到主窗口即可快速与 AI 交互。

## 使用方式

### 紧凑模式

悬浮窗默认以紧凑模式显示（300x60px），呈胶囊形状浮于桌面，包含：

- 应用图标
- 快捷操作入口
- 拖拽手柄

### 展开模式

鼠标悬停在悬浮窗上时，窗口会自动展开（300x400px），显示快捷操作菜单：

- 语音通话
- 屏幕共享
- 会议录制
- 截图问答
- 实时双语字幕

### 打开主窗口

点击悬浮窗可以快速唤起主应用窗口。

## 特性

- **始终置顶** - 悬浮窗始终显示在其他窗口之上
- **可拖拽** - 通过拖拽手柄将悬浮窗移动到屏幕任意位置
- **动画过渡** - 紧凑模式和展开模式之间有平滑的动画效果
- **透明背景** - 窗口使用透明背景，融入桌面环境
