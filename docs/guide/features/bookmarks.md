# 收藏功能

Super Client R 提供消息收藏功能，方便保存和管理对话中的重要信息。

## 收藏消息

在聊天界面中，可以将任意消息添加到收藏：

1. 将鼠标悬停在消息上
2. 点击收藏图标
3. 可选：添加备注说明
4. 消息即被保存到收藏列表

收藏的消息会记录原始内容、角色（用户/助手）和时间戳。

## 管理收藏

进入 **收藏** 页面可以查看和管理所有已收藏的消息：

### 查看收藏

收藏列表以卡片形式展示，每张卡片包含：

- 消息内容
- 发送角色标识（用户/AI 助手）
- 收藏时间
- 备注信息（如有）

### 搜索收藏

使用页面顶部的搜索栏，可以按消息内容或备注文字快速筛选收藏。

### 编辑备注

点击收藏卡片上的编辑按钮，可以修改备注信息，方便日后回忆收藏原因。

### 删除收藏

点击收藏卡片上的删除按钮，可以移除不再需要的收藏。

## 导出收藏

支持将所有收藏导出为 JSON 文件：

1. 在收藏页面点击 **导出** 按钮
2. 选择保存位置
3. 文件将以带日期戳的文件名保存

导出的数据包含所有收藏消息的完整信息，方便备份和迁移。
