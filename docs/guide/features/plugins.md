# 插件系统

Super Client R 提供完整的插件系统，支持通过插件扩展应用功能，包括命令、视图、主题等多种扩展点。

## 插件市场

### 浏览插件

进入 **插件** 页面即可浏览可用的插件列表，每个插件展示：

- 插件名称和图标
- 简短描述
- 作者信息
- 版本号
- 分类标签

### 安装插件

1. 在插件市场中找到目标插件
2. 点击 **安装** 按钮
3. 等待安装完成
4. 插件自动进入已安装状态

## 插件管理

### 插件状态

插件有以下生命周期状态：

| 状态 | 说明 |
|------|------|
| installing | 正在安装 |
| installed | 已安装（未激活） |
| activating | 正在激活 |
| active | 已激活（运行中） |
| deactivating | 正在停用 |
| inactive | 已停用 |
| error | 出错 |
| uninstalling | 正在卸载 |

### 启用/停用

- 点击插件的 **启用** 按钮激活插件
- 点击 **停用** 按钮关闭插件功能
- 停用后插件仍然保留在本地，可以随时重新启用

### 卸载

点击 **卸载** 按钮可以完全移除插件。

## 插件功能

插件可以贡献以下扩展点：

### 命令 (Commands)

插件可以注册自定义命令，通过快捷键或命令面板触发执行。

### 视图 (Views)

插件可以添加自定义视图面板，丰富应用界面。

### 主题 (Themes)

插件可以提供自定义主题，包括 Ant Design Token 配置和 Markdown 渲染样式。

### 快捷键 (Keybindings)

插件可以注册平台相关的快捷键绑定，支持 Mac 和 Windows 不同配置。

### 配置项 (Configuration)

插件可以声明配置属性，用户可在设置中调整。

## 内置插件

### Prompt Templates

内置的 Prompt Templates 插件提供 AI 提示词模板，帮助用户快速生成高质量的 AI 交互提示。

## 开发者模式

插件系统支持开发者模式，方便开发和调试自定义插件。
