# 文件附件

Super Client R 支持在对话中上传和管理文件附件，让 AI 能够理解和处理文件内容。

## 支持的文件类型

| 类型 | 说明 | 示例 |
|------|------|------|
| 图片 (Image) | 图片文件 | PNG、JPG、GIF、WebP |
| 文档 (Document) | 文档文件 | PDF、Word、TXT、Markdown |
| 代码 (Code) | 代码文件 | JS、TS、Python、Java 等 |
| 音频 (Audio) | 音频文件 | MP3、WAV |
| 视频 (Video) | 视频文件 | MP4、WebM |
| 压缩包 (Archive) | 压缩文件 | ZIP、RAR、7z |
| 其他 (Other) | 其他格式 | 任意文件 |

## 上传文件

在聊天界面中，可以通过以下方式添加附件：

1. 点击输入框旁的 **附件** 按钮
2. 在文件选择对话框中选择文件
3. 文件会显示上传进度
4. 上传完成后，附件会显示在输入框下方

## 附件管理

### 查看附件

上传的附件会显示文件信息：

- 文件名和原始文件名
- 文件大小
- 文件类型和 MIME 类型
- 上传时间
- 图片类型附件会显示缩略图预览

### 关联对话

附件可以与特定的对话和消息关联：

- 在发送消息时附带文件，附件自动关联到该消息
- AI 可以读取附件内容并基于文件内容进行回答

### 打开附件

点击附件可以使用系统默认应用打开文件。

### 删除附件

可以单独删除不需要的附件，也支持批量选择删除。

## 存储说明

- 附件以唯一 ID 存储在本地
- 保留原始文件名信息
- 图片类型自动生成缩略图
- 支持 Data URL 格式的预览
