# 介绍

Super Client R（超级客户端）是一款功能强大的 AI 桌面客户端应用程序，基于 Electron 构建，集成了多种先进的 AI 技术和协议。

## 什么是 Super Client R？

Super Client R 是一个开源的 AI 客户端，旨在为用户提供统一、高效的 AI 交互体验。它支持多种 AI 模型、协议和扩展机制，让用户能够灵活地使用各种 AI 功能。

## 核心特性

### 🤖 AI 聊天

- 基于 Claude SDK 的智能对话
- 支持流式响应，实时显示 AI 回复
- 多轮对话和上下文记忆
- 多种聊天模式（直接、Agent、技能、MCP）

### 🧠 模型管理

- 支持 24+ 模型供应商（OpenAI、Anthropic、DeepSeek、Google Gemini、Ollama 等）
- 统一的供应商配置和管理界面
- 连接测试和延迟检测
- 按模型能力分类（视觉、推理、工具调用等）
- 详见 [模型管理](./features/models)

### 🔌 MCP 协议支持

- 完整的 Model Context Protocol 实现
- 连接各种 AI 模型和工具
- 动态工具发现和调用
- 支持自定义 MCP 服务器

### 🔍 联网搜索

- 内置 10 种搜索引擎（智谱、Tavily、SearXNG、Exa、Google、Bing、百度等）
- 灵活的搜索引擎配置和管理
- AI 自动判断何时需要联网搜索
- 详见 [联网搜索](./features/search)

### 🧩 技能系统

- 可扩展的技能架构
- 支持自定义工具和工作流
- 技能市场，方便发现和安装
- 版本管理和自动更新

### 🔧 插件系统

- 完整的插件生命周期管理（安装/启用/激活/停用/卸载）
- 命令注册与执行系统
- 内置插件市场，一键安装
- 插件隔离存储（Memento API）
- 支持主题插件和 Markdown 渲染主题
- 内置 Prompt Templates 插件
- 详见 [插件系统](./features/plugins) 和 [插件开发指南](../PLUGIN_DEVELOPMENT)

### 📁 工作区

- 4 种工作区类型（个人、工作、项目、临时）
- 每个工作区独立配置默认模型、系统提示词、温度等
- 工作区导入/导出
- 详见 [工作区](./features/workspaces)

### ⭐ 收藏功能

- 收藏对话中的重要消息
- 添加备注和搜索筛选
- 导出收藏为 JSON
- 详见 [收藏功能](./features/bookmarks)

### 📎 文件附件

- 支持多种文件类型（图片、文档、代码、音视频等）
- 附件与对话/消息关联
- 图片缩略图预览
- 详见 [文件附件](./features/attachments)

### 💬 悬浮窗

- 桌面悬浮胶囊窗口
- 快速访问 AI 功能
- 始终置顶、可拖拽
- 详见 [悬浮窗](./features/float-widget)

### ⌨️ 快捷键

- 14 个默认快捷键覆盖全局、聊天、导航操作
- 完全可自定义，支持冲突检测
- 跨平台支持（macOS / Windows / Linux）
- 详见 [快捷键](./features/shortcuts)

### 🌐 HTTP API

- 内置 Koa HTTP 服务器
- 提供 RESTful API 接口
- 支持跨域访问
- Swagger 文档自动生成

### 🎨 精美 UI

- 基于 React + Ant Design
- Tailwind CSS 样式系统
- 深色/浅色主题切换
- 响应式布局设计

### 🌍 国际化

- 完整的 i18n 支持
- 轻松切换多语言
- 支持新增语言包

### 🔄 自动更新

- 自动检查和下载新版本
- 下载进度显示
- 退出时自动安装
- 详见 [自动更新](./features/auto-update)

### 📋 日志查看器

- 独立窗口的日志查看器
- SQLite 结构化存储
- 多维度筛选和统计
- 详见 [日志查看器](./features/log-viewer)

## 适用场景

- **AI 助手**：日常使用 Claude 等 AI 模型进行对话
- **开发工具**：通过 MCP 协议连接各种开发工具
- **自动化工作流**：使用技能系统构建自动化流程
- **API 服务**：通过 HTTP API 集成到其他应用
- **多模型管理**：统一管理多个 AI 供应商的模型
