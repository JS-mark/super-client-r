# 介绍

Super Client R（超级客户端）是一款功能强大的 AI 桌面客户端应用程序，基于 Electron 构建，集成了多种先进的 AI 技术和协议。

## 什么是 Super Client R？

Super Client R 是一个开源的 AI 客户端，旨在为用户提供统一、高效的 AI 交互体验。它支持多种 AI 模型、协议和扩展机制，让用户能够灵活地使用各种 AI 功能。

## 核心特性

### 🤖 AI 聊天

- 基于 Claude SDK 的智能对话
- 支持流式响应，实时显示 AI 回复
- 多轮对话和上下文记忆
- 多种聊天模式（直接、Agent、技能、MCP）

### 🔌 MCP 协议支持

- 完整的 Model Context Protocol 实现
- 连接各种 AI 模型和工具
- 动态工具发现和调用
- 支持自定义 MCP 服务器

### 🧩 技能系统

- 可扩展的技能插件架构
- 支持自定义工具和工作流
- 技能市场，方便发现和安装
- 版本管理和自动更新

### 🌐 HTTP API

- 内置 Koa HTTP 服务器
- 提供 RESTful API 接口
- 支持跨域访问
- Swagger 文档自动生成

### 🎨 精美 UI

- 基于 React + Ant Design
- Tailwind CSS 样式系统
- 深色/浅色主题切换
- 响应式布局设计

### 🌍 国际化

- 完整的 i18n 支持
- 轻松切换多语言
- 支持新增语言包

## 适用场景

- **AI 助手**：日常使用 Claude 等 AI 模型进行对话
- **开发工具**：通过 MCP 协议连接各种开发工具
- **自动化工作流**：使用技能系统构建自动化流程
- **API 服务**：通过 HTTP API 集成到其他应用
